Project Overview

What this model actually does

This system estimates how risky a building may be during an earthquake using important construction, design, and location-related features. Instead of relying only on manual judgment, the model provides a faster and data-driven indication of possible structural vulnerability.

🏗️

Structural Analysis

The application studies building properties like number of floors, age, height, foundation type, roof type, and floor material to estimate how these characteristics influence earthquake damage risk.

📊

Smart Prediction

A trained machine learning model processes the user inputs and classifies the building into a meaningful risk level, helping users understand whether the building appears less vulnerable or highly exposed.

🛡️

Awareness & Preparedness

This project can support educational demonstrations, research explanation, and awareness about building safety by showing how different structural decisions can influence predicted earthquake risk.
